## BP17 Motor Controller Data Logger Node

This is considered to be one of the Node in BP17 Telemetry Layer 1

It acts as a local sd card logger , and a transport gateway to the main SBC, publishing data using microROS platformio 

BAMOCAR-PG-D3-700-400

## How to use this project

The project must be build in Linux environment, and it would be best if you pull our docker image

### Build in this docker container
```bash
docker image pull xx

docker run xx
```
Clone this repo to your workspace <br> 
`git clone https://github.com/To-en/microROS-BAMOcar.git`
